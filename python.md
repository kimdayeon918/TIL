### 데이터 타입

문자형 - str

논리형 - bool

### 대입연산자 =

왼쪽에 명시된 변수에 해당 값을 대입

변수는 해당 값을 가지게 됨

```python
a = 17
b = 9.18
c = "김다연"
```
